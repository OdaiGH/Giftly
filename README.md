# Giftly

Giftly is a delivery platform connecting customers with couriers for seamless package delivery services.

## Team Members

- **Abdulrahman Aljuraisy** - Founder
- **Muhannad Alhusaini** - Co-Founder
- **Odai Alghamdi** - CTO

## Project Structure

- `frontend/` - React Native mobile application
- `backend/` - Server-side API and services

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/OdaiGH/Giftly.git
   cd Giftly
   ```

2. Install dependencies for frontend:
   ```bash
   cd frontend
   npm install
   ```

3. Set up environment variables:
   - Copy `.env.local` and configure your API keys

4. Run the development server:
   ```bash
   npm run dev
   ```

## Features

- Customer and courier role selection
- Real-time chat between customers and couriers
- Order tracking and invoice generation
- Profile management
- Budget and city selection for deliveries
